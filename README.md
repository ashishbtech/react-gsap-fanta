# Fanta Interactive Landing Page 🍊

A high-performance, interactive product landing page built to explore seamless scroll-driven animations and modern frontend architecture.

This project leverages **GSAP ScrollTrigger** to pin and animate a central product (the Fanta can) across multiple sections—creating a cinematic, almost 3D scrolling experience.

### 💡 Project Notes

**Inspiration**
The animation style and layout were inspired by a UI concept I discovered on YouTube Shorts, which sparked the idea for a scroll-driven product experience.

**The Journey**
This project marks my transition into the React ecosystem—my **first project built with React**. After spending months creating complex UI clones using vanilla JavaScript, I built this to translate my animation skills into a component-based architecture.

**Looking Ahead**
While this is an early React project and not perfect, it represents a major learning milestone. I’m excited to continue refining my skills and building more optimized, scalable, and interactive applications.

### 🛠️ Running Locally

```bash
# Clone the repository
git clone https://github.com/ashishbtech/react-gsap-fanta.git

# Navigate into the project
cd react-gsap-fanta

# Install dependencies
npm install

# Start development server
npm run dev
```
